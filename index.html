media-space by praveenvijayan

Media space is a jQuery gallery plugin to display mixed media in once place. Images, video and flash can be displayed in a single place.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/praveenvijayan/media-space
