Dependencies

  • jQuery
  • jquery.mediaspace.js
  • mediaspace.css
 

Usage

Basic usage

$('#media-container ul').mediaspace();

With options

$('#media-container ul').mediaspace({
	container:'media-box',
	fit:false,
	title:true
});
